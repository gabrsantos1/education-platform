
# Education Platform


## Inicializando a aplicação

### Passo 1: Iniciando o projeto:

Inicialize o projeto:

```bash
  npm init -y
```

Instale as dependências:

```bash
  npm install express 
  npm install mysql2 
  npm install cors
```

### Passo 2: Rodando a aplicação:

Para executar a aplicação utilize:

```bash
  node backend.js 
```
